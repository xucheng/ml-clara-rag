# ============================================================================
# CLaRa Dependencies - Installation Guide
# ============================================================================
#
# This project has TWO separate dependency files for different use cases:
#
# 1️⃣  For TRAINING on Google Colab:
#     pip install -r requirements-training.txt
#
# 2️⃣  For LOCAL DATA PIPELINE (extracting from documents, synthesizing data):
#     pip install -r requirements-data-pipeline.txt
#
# 3️⃣  For FULL INSTALLATION (both training and data pipeline):
#     pip install -r requirements-training.txt
#     pip install -r requirements-data-pipeline.txt
#
# ============================================================================
# Why separate files?
# ============================================================================
#
# - Training dependencies are minimal and optimized for Colab
# - Data pipeline requires document extraction libraries (not needed for training)
# - This separation reduces installation time and potential conflicts
#
# ============================================================================
# Quick Start Examples
# ============================================================================
#
# Example 1: Train on Colab with existing data
# ---------------------------------------------
# pip install -r requirements-training.txt
# # Then use training_colab_complete.ipynb
#
# Example 2: Process your own documents locally, then train
# ----------------------------------------------------------
# pip install -r requirements-data-pipeline.txt
# bash scripts/run_data_pipeline.sh
# # Upload generated .jsonl files to Colab, then train
#
# Example 3: Full local development (training + data processing)
# ----------------------------------------------------------------
# pip install -r requirements-training.txt
# pip install -r requirements-data-pipeline.txt
#
# ============================================================================
